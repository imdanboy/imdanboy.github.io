## 5 {Exam3=F}       => {Exam1=C}       0.1304348 0.8333333  1.742424

How to find the best association rule to fill in missing attributes

I choose the rule in this priority

  1. more specific rule, in my R code n_attr should be higher ex) {Exam1=A, Exam2=B} => {Exam3=B} is better than {Exam1=A} => {Exam3=B}
  2. higher support
  3. higher confidence
  4. if rule doesn’t exist, just choose mode value in that column

5. Conclusion

Despite of small dataset size (about a hundred), I could observe similar results described at reference paper.
With appropriate support and confidence, ‘car asap classifier’ could predict student’s failure better than pessimistic, and almost same results as optimistic!
